# OpenBTS-UMTS reloaded 2023

OpenBTS-UMTS reloaded for 2023. This fork extends changes from @EurecatSecurity make it compatiblee with latest UHD drivers, several fixes to work on Ubuntu 22.04.

## Compatible devices

* USRP
  * B200/B210/B205-mini*
  * X300/X310
  * N200/N210
  * USRP2
